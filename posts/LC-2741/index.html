🔗 🟡 2741. Special Permutations 2021

tags: Weekly Contest 350 狀態壓縮(Bitmask) 動態規劃(DP) 位運算(Bit Manipulation) 狀壓DP

題意

給定一個包含 nnn 個 不同 正整數的整數陣列 numsnumsnums ，下標從 000 開始。如果 numsnumsnums 的一個排列 perm\text{perm}perm 滿足以下條件，則稱其為特殊排列：

  • 對於 0≤i<n−10 \leq i < n - 10≤i<n−1 的下標 iii ，滿足 perm[i] mod perm[i+1]=0∨perm[i+1] mod perm[i]=0\text{perm}[i] \bmod \text{perm}[i+1] = 0 \lor \text{perm}[i+1] \bmod \text{perm}[i] = 0perm[i]modperm[i+1]=0∨perm[i+1]modperm[i]=0 。

返回特殊排列的總數量，由於答案可能很大，請將其對 109+710^9 + 7109+7 取模後返回。

約束條件

  • 2≤n≤142 \leq n \leq 142≤n≤14
  • 1≤nums[i]≤1091 \leq \text{nums}[i] \leq 10^91≤nums[i]≤109

思路：狀壓 DP ：排列型－相鄰相關

令 dfs(S,i)dfs(S, i)dfs(S,i) 表示當前已選擇的集合為 SSS，且最後一個選擇的數字是 nums[i]nums[i]nums[i] 時，滿足條件的特殊排列的數量。則我們可以得到狀態轉移方程：

  • dfs(S,i)=∑j=0n−1dfs(S∪{j},j)dfs(S, i) = \displaystyle\sum_{j=0}^{n-1} dfs(S \cup \{j\}, j)dfs(S,i)=j=0∑n−1​dfs(S∪{j},j) ，其中 jjj 是下一個選擇的數字的下標。
    • 需滿足 j∉Sj \notin Sj∈/S 且 nums[i] mod nums[j]=0∨nums[j] mod nums[i]=0nums[i] \bmod nums[j] = 0 \lor nums[j] \bmod nums[i] = 0nums[i]modnums[j]=0∨nums[j]modnums[i]=0 。
  • 遞迴終點為 S=US = US=U ，即所有數字都已經被選擇，其中 UUU 為 宇集合(Universal Set) 。

而注意到 ∣S∣=n≤14|S| = n \leq 14∣S∣=n≤14 ，因此我們可以使用一個 141414 位的二進制數字來表示 SSS ，其中第 iii 位為 111 表示 iii 已經被選擇，為 000 表示未被選擇， 狀態壓縮 成一個整數，並將集合的運算轉換為位運算(Bit Manipulation)。

  • 由於添加到集合的操作會使該位從 000 變為 111 ，因此可以使用 s∣(1<<j)s | (1 << j)s∣(1<<j) 來表示添加 jjj 到 SSS 中。
  • 宇集合 UUU 可以表示為 u=(1<<n)−1u = (1 << n) - 1u=(1<<n)−1 。

由於會存在一些重複的子問題，因此我們可以使用 動態規劃(DP) 來避免重複計算，這裡使用了 Top-Down 的方式進行遞迴計算，並使用 記憶化搜尋(Memoization) 來保存已經計算過的結果。

初始化時可以考慮第一個選擇的數字，即 dfs(1<<i,i)dfs(1 << i, i)dfs(1<<i,i) ，其中 iii 是 numsnumsnums 的下標，將其所有的結果相加即為答案。

複雜度分析

  • 時間複雜度：O(n2⋅2n)\mathcal{O}(n^2 \cdot 2^n)O(n2⋅2n)，其中 nnn 是 numsnumsnums 的長度。總共有 n⋅2nn \cdot 2^nn⋅2n 種狀態，每種狀態需要 O(n)\mathcal{O}(n)O(n) 的時間進行轉移計算。
  • 空間複雜度：O(n⋅2n)\mathcal{O}(n \cdot 2^n)O(n⋅2n)，即狀態數量。
